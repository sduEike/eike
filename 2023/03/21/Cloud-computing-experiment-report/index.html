实验目的

实验目的：熟悉个人博客系统的搭建。

具体包括：注册Github账号，搭建Hexo环境并实现个人博客搭建，撰写实验报告。

参考方案：注册Github账号，搭建Hexo环境并实现个人博客搭建，撰写实验报告。

实验步骤与内容

*注册Github账号*

访问 Github 官网 ，点击右上角的 “Sign up” 按钮

输入用户名，邮箱和密码，然后单击 “Create an account” 按钮

完成验证后，将进入Github的欢迎页面

*搭建Hexo环境*

打开终端或命令行工具，并安装Node.js和Git

执行以下命令来安装Hexo：

npm install -g hexo-cli

Git bash使用以下命令来创建一个新的Hexo博客：

hexo init

进入新创建的博客目录并安装依赖：

cd myblog

npm install

*实现个人博客搭建*

在Github上创建一个新的仓库，用于托管博客。请注意，仓库名称必须是 username.github.io，其中 username 是Github用户名

在Hexo博客目录中，找到 _config.yml 文件并编辑它。将以下内容替换为Github用户名和仓库名称：

执行以下命令来生成博客：

hexo generate

执行以下命令来部署博客：

hexo deploy

访问 https://username.github.io/ 即可查看博客

已经成功搭建了个人博客！可以根据自己的需要编辑Hexo博客的配置文件和主题，以实现更好的博客体验。

*美化*

可通过下载主题更改主题，更改_config.yml文件修改信息
